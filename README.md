JoyJava
=======

This is a Java implementation of Joy.

Setup
=====
JoyJava should be in the same parent directory as the following prerequisite projects;

    JavaCommon
    Joy

Protocol Buffers

    cd <path/to/Joy>
    ./build.sh --java_out=lite:<path/to/JoyJava>/source/

Build
=====

    ./build.sh
